# Banner Generator

Генератор вебинарных баннеров под цвета любого сайта. Создаёт баннеры для Telegram и Email за пару минут — без дизайнера, без платных сервисов.

Дипломный проект курса Claude Code.

---

## Что делает

- Извлекает цвета автоматически по URL сайта клиента
- Генерирует баннеры двух форматов: **ТГ 1080×1080** и **Email 1080×420**
- Три типа баннеров: анонсные с текстом, анонсные с плашкой, статусные с иконкой
- Редактор каждого баннера: перемещение и масштаб картинки/текста, замена фото
- Скачивание всех баннеров одним ZIP-архивом

---

## Шаблоны баннеров

**Анонсные** (фото человека + текст вебинара):
- Без даты
- С датой
- «Уже завтра»
- «Уже сегодня»
- «Начинаем» / «Мы в эфире» — без заголовка, только плашка

**Статусные** (иконка + плашки):
- 15 минут / 5 минут / Час до старта
- Повтор / Бонус / Выгода / Главное

## Быстрый старт

```bash
git clone <url репозитория>
cd banner-generator
npm install
npm run dev
```

Открыть в браузере: **http://localhost:3000**

> Требуется Node.js 18+

---

## Как пользоваться

1. **Введи URL сайта** → цвета извлекутся автоматически (или настрой вручную)
2. **Заполни текст**: категория, заголовок, подзаголовок, дата
3. **Загрузи фото** (для анонсных баннеров) и **иконки** (для статусных)
4. Нажми **«Генерировать баннеры»**
5. Переключай табы ТГ / Email / Все
6. Редактируй каждый баннер кнопкой **✎**
7. Скачивай по одному или **«↓ Скачать ZIP»**

---

## Редактор баннера

Открывается по кнопке ✎ на каждой карточке.

| Режим | Что можно делать |
|---|---|
| Картинка | Перетащить фото, изменить масштаб, заменить файл |
| Текст | Перетащить текстовый блок, изменить размер |
